ポートフォリオ

ソート比較プログラム
使用言語: Python

ユーザが設定した要素数を最良の場合・最悪の場合に並び替えて，各種ソートの処理時間を比較することが出来ます．ソートの種類には，
挿入/選択/バブル/クイック/マージ
があります．

書籍記録プログラム
使用言語: Python

このアプリは書籍を簡単に記録できるコンソールアプリです．記録は指定されたテキストファイルに保存されます．また，記録に対して，
追加/表示/編集/探索/並べ替え/削除
の操作をすることが出来ます．

スケジュール管理アプリ
使用言語: HTML,CSS,Javascript,PHP,SQLite

このアプリはスケジュールを管理するWebアプリケーションです．1955年から2025年までの祝日が記載された月単位のカレンダーを表示し，クリックした日の予定を入力するとデータベースに予定が格納され，いつでも確認することが出来ます．
